# Miro Tasks Plugin

Track your coding progress on a Miro board. Automatically syncs todos and task status to visual cards.

## Components

| Type     | Details |
|----------|---------|
| Commands | `enable`, `disable`, `status` |
| Hooks    | `stop` — sync tasks when session ends |
| Scripts  | Task tracking shell scripts |

## Prerequisites

- Miro MCP must be installed and configured.
- A Miro table URL (deep link with `moveToWidget` or `focusWidget`).

## Installation

1. Enable Miro MCP.
2. Run the enable command with a table URL to start tracking.
